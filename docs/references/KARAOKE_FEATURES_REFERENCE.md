# 既存カラオケ機能リファレンス
これは一般的なカラオケシステムに搭載されている機能のリストです。
MinKara の開発において、どの機能を実装すべきか、あるいは実装しないか（スコープ外とするか）を判断するための参考資料として使用します。

- **楽曲検索**

    - [ ] ジャンル検索
    - [ ] 新曲／ランキング検索
    - [ ] 年代別検索
    - [ ] アニメ・ボカロ・K-POP 等カテゴリ検索
- **楽曲予約**

    - [ ] 予約削除
    - [ ] 連続予約（複数曲一括）
- **再生制御**
    - [ ] 再生／停止

    - [ ] 前奏スキップ
    - [ ] 間奏スキップ
- **キー・テンポ調整**
    - [ ] キー変更（±n）
